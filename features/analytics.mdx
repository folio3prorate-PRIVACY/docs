# Analytics & Insights

Gain deep insights into your digital asset portfolio with Octav's advanced analytics.

## Performance Metrics

Track comprehensive performance metrics:

- **Total Portfolio Value** - Current value across all assets
- **24h Change** - Daily performance tracking
- **All-Time P&L** - Total profit and loss
- **ROI** - Return on investment calculations
- **IRR** - Internal rate of return

## Visualization Tools

### Interactive Charts
- Line charts for portfolio value over time
- Bar charts for asset allocation
- Pie charts for diversification analysis
- Candlestick charts for individual assets

### Custom Time Periods
Analyze performance over any time period:
- 24 hours
- 7 days
- 30 days
- 90 days
- 1 year
- All time
- Custom date ranges

Analyze your trading and transaction patterns:
