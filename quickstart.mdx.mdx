### Send your first API request

**Note:** Create an account and generate your API key . \
\
For production, store credentials securely using environment variables 

---

#### 1. Get your API credentials

Sign up at [data.octav.fi](http://data.octav.fi), then navigate to **API Key** to copy your Secret API key.

#### 2. Clone the Quickstart repo

```
git clone --depth=1 https://github.com/octav-labs/octav-sdk && cd octav-sdk/examples/typescript/quickstart
```

#### 3. Run the project

```
npm install && npm run build && npm start
```
