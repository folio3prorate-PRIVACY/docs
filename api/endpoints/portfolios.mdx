# Portfolios API

Create and manage portfolios with multiple wallets and track performance.

## List Portfolios

Get all portfolios in your account.

### Endpoint

```
GET /v1/portfolios
```

### Example Request

```bash
curl https://api.octav.fi/v1/portfolios \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json
{
```

## Create Portfolio

Create a new portfolio.

### Endpoint

```
POST /v1/portfolios
```

### Request Body

```json
{
  "name": "DeFi Portfolio",
  "description": "DeFi protocol positions",
  "wallets": [
    "0xabc123...",
    "0xdef456..."
  ]
}
```

### Response

```json
{
  "data": {
    "id": "portfolio_789",
    "name": "DeFi Portfolio",
    "description": "DeFi protocol positions",
    "created_at": "2024-11-03T12:00:00Z"
  }
}
```

## Update Portfolio

Update portfolio details.

### Endpoint

```
PATCH /v1/portfolios/{portfolio_id}
```

### Request Body

```json
{
  "name": "Updated Portfolio Name",
  "description": "New description"
}
```

## Portfolio Performance

Get performance metrics for a portfolio.

### Endpoint

```
GET /v1/portfolios/{portfolio_id}/performance
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `period` | string | No | Time period (24h, 7d, 30d, 90d, 1y, all) |
| `interval` | string | No | Data interval (hour, day, week) |

### Example Request

```bash
curl "https://api.octav.fi/v1/portfolios/portfolio_123/performance?period=30d&interval=day" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
```

## Delete Portfolio

Remove a portfolio.

### Endpoint

```
DELETE /v1/portfolios/{portfolio_id}
```

### Example Request

```bash
curl -X DELETE https://api.octav.fi/v1/portfolios/portfolio_123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

<Alert type="warning" title="Deletion">
Deleting a portfolio does not delete the wallets or their data. Only the portfolio grouping is removed.
</Alert>
