## Adding Widgets to Your Dashboard

<Steps>
  <Step title="Open Widget Manager" icon="grid-2">
    From your dashboard, click the board menu and select **Add Widget**
  </Step>

  <Step title="Browse Available Widgets" icon="magnifying-glass">
    Search or filter widgets by:
    - **Search bar** - Find widgets by name
    - **Category** - Filter by Overview, Chains, Protocols, Wallet, Chain Filters
    - **License** - Filter by Free, Lite, or Pro
  </Step>

  <Step title="Preview Widget" icon="eye">
    Click on a widget to see its details:
    - Widget name and description
    - Author (Octav Labs)
    - Price
    - License requirement
    - Category
  </Step>

  <Step title="Add to Dashboard" icon="plus">
    Click **Add to Dashboard** to install the widget

    - Free widgets add instantly
    - Paid widgets require purchase or valid license
  </Step>

  <Step title="Arrange & Customize" icon="up-down-left-right">
    Drag to position the widget and resize as needed
  </Step>
</Steps>

<Tip>
  **Try Before You Buy** â€” Preview all widgets to see if they fit your needs before purchasing.
</Tip>

## Widget Search & Filters

<Tabs>
  <Tab title="Search" icon="magnifying-glass">

    **Search Widgets by Name**

    Use the search bar at the top of Widget Manager:
    - Type widget name or keywords
    - Results filter instantly
    - Case-insensitive search
    - Partial matching supported

    **Example Searches:**
    - "portfolio" - Shows Portfolio Summary, Portfolio Timeline, etc.
    - "chart" - Shows all chart-type widgets
    - "chain" - Shows chain-related widgets
    - "networth" - Shows net worth widgets

  </Tab>

  <Tab title="Category Filter" icon="filter">

    **Filter by Category**

    Select from available categories:
    - **All** - Show all widgets
    - **Overview** - Portfolio summaries and analytics
    - **Chain Filters** - Chain selection widgets
    - **Protocols** - DeFi protocol widgets
    - **Chains** - Chain distribution widgets
    - **Wallet** - Wallet-specific widgets

  </Tab>

  <Tab title="License Filter" icon="key">

    **Filter by License Tier**

    Filter widgets by license requirement:
    - **All** - Show all widgets regardless of license
    - **Free** - Show only free widgets
    - **Lite** - Show Lite-tier widgets ($10-50)
    - **Pro** - Show Pro-tier widgets ($25-50)

    <Info>
      **Check Your License** â€” Your active licenses are displayed in the Widget Manager.
    </Info>

  </Tab>
</Tabs>

## Managing Active Widgets

<Tabs>
  <Tab title="View Active Widgets" icon="list-check">

    **See All Installed Widgets**

    The **Active Widgets** tab in Widget Manager shows:
    - All widgets currently on your dashboard
    - Widget name and category
    - Current board location
    - Quick access to settings

    **Actions:**
    - Remove widgets you no longer need
    - Jump to widget location on dashboard
    - View widget settings
    - Reorder widgets in list

  </Tab>

  <Tab title="Configure Widgets" icon="gear">

    **Widget Settings**

    Each widget has configurable options:
    - Display preferences
    - Data filters
    - Time ranges
    - Color schemes
    - Update frequency

    Access settings by clicking the gear icon on any widget.

  </Tab>

  <Tab title="Remove Widgets" icon="trash">

    **Delete from Dashboard**

    Remove widgets you don't use:
    - Click remove in Widget Manager
    - Or click X button on widget itself
    - Widgets can be re-added anytime
    - Removal doesn't affect license

  </Tab>
</Tabs>
