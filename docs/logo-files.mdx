## File Format Guide

<Alert type="tip" title="Which format should I use?">
Choose the right format for your use case.
</Alert>

### SVG (Scalable Vector Graphics)
- **Best for:** Web, digital, scalable applications
- **Pros:** Infinitely scalable, small file size, editable
- **Use when:** Building websites, apps, digital products

### PNG (Portable Network Graphics)
- **Best for:** Web, presentations, quick usage
- **Pros:** Transparent backgrounds, widely supported
- **Use when:** Need raster image with transparency

### JPG/JPEG
- **Best for:** Photography, solid backgrounds
- **Pros:** Small file size, universal support
- **Use when:** Don't need transparency, want smaller files

### EPS (Encapsulated PostScript)
- **Best for:** Print design, professional graphics
- **Pros:** Vector format, print-ready, editable in Adobe
- **Use when:** Sending to print shops, professional design

### PDF (Portable Document Format)
- **Best for:** Print, sharing, archiving
- **Pros:** Preserves quality, universal, embeds fonts
- **Use when:** Sharing with vendors, archiving

## Usage Guidelines

### Minimum Sizes

**Logo with text:**
- Digital: 120px width minimum
- Print: 1 inch width minimum

**Icon only:**
- Digital: 32px minimum
- Print: 0.25 inch minimum

### Clearspace
Maintain clearspace around logos equal to the height of the "O" in Octav.
