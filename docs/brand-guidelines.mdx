# Octav Brand Guidelines

Official brand guidelines for using the Octav name, logo, and visual identity.

## Overview

These guidelines help maintain consistency and quality across all Octav communications, products, and marketing materials.

## Typography

<Tabs>
  <Tab title="Space Grotesk" icon="heading">

    ### Primary Font: Space Grotesk

    Used for headings, titles, and emphasis.

    **Weights:**
    - Light (300)
    - Regular (400)
    - Medium (500)
    - Bold (700)

    **Usage:**
    - H1: 48px, Bold
    - H2: 36px, Bold
    - H3: 28px, Medium
    - H4: 24px, Medium
    - H5: 20px, Medium
    - H6: 16px, Bold

  </Tab>

  <Tab title="Geist" icon="text">

    ### Secondary Font: Geist

    Used for body text, UI elements, and data display.

    **Weights:**
    - Regular (400)
    - Medium (500)
    - Semibold (600)

    **Usage:**
    - Body Large: 18px, Regular
    - Body: 16px, Regular
    - Body Small: 14px, Regular
    - Caption: 12px, Regular
    - Button: 16px, Medium

  </Tab>

  <Tab title="Code Examples" icon="code">

    ### Font Pairing Examples

    ```css
    /* Heading + Body */
    h1 {
      font-family: 'Space Grotesk', sans-serif;
      font-weight: 700;
      font-size: 48px;
    }

    p {
      font-family: 'Geist', sans-serif;
      font-weight: 400;
      font-size: 16px;
      line-height: 1.6;
    }
    ```

  </Tab>

### Logo Clearspace

<Info>
  Maintain clearspace around the logo equal to the height of the "O" in Octav.
</Info>

**Minimum clearspace:**
- All sides: Height of "O" letter
- Never place other elements within clearspace
- Applies to all logo variations

### Logo Guidelines

<AccordionGroup>
  <Accordion title="Do's ✅" icon="check" defaultOpen>
    - Use official logo files
    - Maintain aspect ratio
    - Ensure sufficient contrast
    - Use on appropriate backgrounds
    - Respect clearspace
  </Accordion>

  <Accordion title="Don'ts ❌" icon="xmark">
    - Stretch or distort the logo
    - Change logo colors
    - Add effects (drop shadow, gradient, etc.)
    - Rotate the logo
    - Place on busy backgrounds
    - Use low-resolution files
    - Recreate or redraw the logo
  </Accordion>
</AccordionGroup>

### Writing Style

<Tabs>
  <Tab title="Be Clear" icon="eye">

    - Use simple, direct language
    - Avoid jargon when possible
    - Explain technical concepts simply
    - Get to the point quickly

  </Tab>

  <Tab title="Be Helpful" icon="hand-holding-heart">

    - Anticipate user needs
    - Provide actionable guidance
    - Offer solutions, not just information
    - Include examples when helpful

  </Tab>

    - Write conversationally
    - Show empathy
    - Use active voice
    - Acknowledge challenges

  </Tab>
</Tabs>

## Cards & Containers

<Tabs>
  <Tab title="Card Styles" icon="square">

    ### Default Card

    - Background: White (light) / #1A202C (dark)
    - Border: 1px solid #E2E8F0 (light) / #2D3748 (dark)
    - Border radius: 12px
    - Padding: 24px
    - Shadow: Subtle (0 2px 8px rgba(0,0,0,0.08))

    ### Elevated Card

    - Same as default card
    - Shadow: Medium (0 4px 16px rgba(0,0,0,0.12))
    - Usage: Important content, modals

    ### Outlined Card

    - Background: Transparent
    - Border: 2px solid
    - No shadow
    - Usage: Empty states, placeholders

  </Tab>
