## Generating Reports

### Portfolio Snapshot Report

<Steps>
  <Step title="Navigate to Reports" icon="file">
    Go to the Reports page from the main navigation
  </Step>

  <Step title="Click Create Snapshot" icon="camera">
    Select "Create" under Portfolio Snapshot Report
  </Step>

  <Step title="Select Addresses or Bundles" icon="wallet">
    Choose the addresses or bundles to include in your snapshot
  </Step>

  <Step title="Choose Export Format" icon="download">
    Select WEB, PDF, or CSV format
  </Step>

  <Step title="Generate Snapshot" icon="bolt">
    Click "Generate Snapshot" to create your report
  </Step>
</Steps>

### Transaction Report

<Steps>
  <Step title="Navigate to Reports" icon="file">
    Go to the Reports page from the main navigation
  </Step>

  <Step title="Click Create Transaction Report" icon="list">
    Select "Create" under Transactions Report
  </Step>

  <Step title="Select Addresses or Bundles" icon="wallet">
    Choose which addresses or bundles to include
  </Step>

  <Step title="Apply Filters (Optional)" icon="filter">
    Use the available filters to narrow down transactions:
    - Networks
    - Protocols
    - Categories
    - Assets
    - Transaction Types
    - Date Range
  </Step>

  <Step title="Configure Options" icon="gear">
    - Choose export format (WEB or CSV)
    - Toggle "Hide spam transactions" if desired
    - Review transaction count
  </Step>

  <Step title="Generate Report" icon="bolt">
    Click "Generate Report for X txs" to create your report
  </Step>
</Steps>

### Asset Variation Report

<Steps>
  <Step title="Navigate to Reports" icon="file">
    Go to the Reports page from the main navigation
  </Step>

  <Step title="Click Create Asset Report" icon="chart-line">
    Select "Create" under Asset Variation Report
  </Step>

  <Step title="Select Addresses or Bundles" icon="wallet">
    Choose which addresses or bundles to include
  </Step>

  <Step title="Set Date Range" icon="calendar">
    Select the time period for comparison (e.g., compare two months)
  </Step>

  <Step title="Apply Filters (Optional)" icon="filter">
    Use the available filters:
    - Networks
    - Protocols
  </Step>

  <Step title="Configure Options" icon="gear">
    - Choose export format (WEB or CSV)
    - Toggle "Hide spam assets" if desired
    - Review asset count
  </Step>

  <Step title="Generate Report" icon="bolt">
    Click "Generate Report for X assets" to create your report
  </Step>
</Steps>

## Managing Your Reports

### Your Reports Section

<Info>
  All generated reports are saved and accessible in the "Your Reports" section.
</Info>

**Features:**
- Search through your reports
- Filter by report type (Snapshot, Transaction, Asset)
- Filter by addresses or bundles
- Filter by date range
- View report metadata

**Report Information Displayed:**
<CardGroup cols={2}>
  <Card title="Report Details" icon="info-circle">
    - Creation date
    - Report name
    - Report type
    - Associated addresses/bundles
  </Card>

  <Card title="Scheduled Reports" icon="calendar-check">
    - Active/inactive status
    - Next snapshot date
    - Last snapshot date
    - Number of addresses included
  </Card>
</CardGroup>

### Filtering Your Reports

Use the built-in filters to quickly find reports:

- **Search Bar** - Search reports by name
- **Address/Bundle Filter** - Select specific addresses or bundles
- **Date Picker** - Filter by creation date
- **Report Type Tabs** - Toggle between Snapshot, Transaction, and Asset reports

### Scheduled Reports

<Info>
  Create recurring reports that generate automatically on a schedule.
</Info>

**Scheduled Report Features:**
- Set as Active or Inactive
- Automatic generation at specified intervals
- View next scheduled generation date
- Access all historical snapshots
- Manage multiple scheduled reports per bundle

**Example Use Case:**
Create a monthly snapshot of your "Hot Wallet 1" bundle that generates automatically on the 1st of each month.
